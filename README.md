# 🚀 100FreeAI: The Ultimate Free AI Directory

A curated, hand-picked directory of **100+ elite AI tools** specifically chosen for Writing, Design, and Video Generation. Built for creators, by a creator.

[Visit the Live Site](https://100freeai.netlify.app/)

---

## 🌟 The Philosophy: The "Human Touch"
In a world flooded with AI, finding the *right* tool is hard. 100FreeAI is built on the philosophy that tools should be:
1. **Accessible:** Focus on Free and "Freemium" tiers.
2. **Beginner-Friendly:** Curated for those just starting their AI journey.
3. **High Utility:** Only tools that actually deliver results make the list.

## 🛠️ Key Features
* **100+ Curated Tools:** Categorized for easy navigation.
* **Real-Time User Ratings:** Powered by **Firebase** so the community decides what's best.
* **Security First:** Protected by **Firebase App Check** and **reCAPTCHA** to ensure authentic community feedback.
* **SEO Optimized:** Built with clean HTML5/CSS3 for maximum discoverability.

## 💻 Tech Stack
- **Frontend:** HTML5, CSS3 (Modern Flexbox/Grid), JavaScript (Vanilla)
- **Backend:** Firebase Realtime Database
- **Security:** Firebase App Check & Google reCAPTCHA Enterprise
- **Hosting:** Netlify
